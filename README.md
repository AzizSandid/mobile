## üëã Quick start

Clone the repo to your machine and install deps :

```sh
git clone https://github.com/user/repo-name

cd ./repo-name

pnpm install

pnpm prebuild

```

To run the app on ios

```sh
pnpm ios
```

To run the app on Android

```sh
pnpm android
```

To run the app on Expo go

```sh
npx expo start -c
```
